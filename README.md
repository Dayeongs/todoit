
# TODOIT

> 개인별 취미나 여가시간에 대한 관심도가 높아지는 시기입니다. <br>
TODOIT에서는 취미와 관심사를 바탕으로 많은 사람들이 함께 취미생활을 공유하고 즐길 수 있도록 도와줍니다. <br>
원데이 클래스, 소셜 모임 등을 검색하여 참여 및 결제까지 한번에 이용할 수 있습니다.
